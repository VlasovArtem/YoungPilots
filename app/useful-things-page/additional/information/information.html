Полезняшки (Useful Things)

Это самый интересный контент подкаста. Ни одна из записей подкаста не обходится без интересных ссылок, новых фрейворков или просто полезных вещей для программиста. Именно здесь мы собрали все полезные ссылки в одном месте для вашего удобства.

Полезная информация

Используйте поиск
Вы можете искать по Name, Description, Tag и Podcast Number
Кликайте по Tag
После клика по Tag он автоматически появиться в строке поиска и вы увидите все полезняшки с тем же тагом.
Кликайте по Podcast
После клика по Podcast он автоматически появиться в строке поиска и вы увидите все полезняшки из выбранного выпуска.
